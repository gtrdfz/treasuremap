# La carte aux trésors

Auteur: Gautier Defossez

## Prérequis

- maven
- java 17

## Build project

```
mvn clean package
```

## Utilisation

Dans `/target`

```
java -jar treasuremap-1.0.0-jar-with-dependencies.jar ..\examples\input1.txt
```

Le chemin du fichier de résultat sera indiqué dans les logs de la console.
